Create `includes/destinations-data.php` as a PHP file that declares `$destinations` — a PHP associative array keyed by the 6 valid slugs. Every destination entry must have exactly these keys:

- `name` (string, display name in Italian)
- `hero_image` (string, Unsplash direct URL — high quality landscape photo matching the destination)
- `intro_paragraphs` (array of exactly 3 strings — emotionally compelling Italian editorial copy, good-to-ship quality, each paragraph 2-4 sentences, evocative and premium in tone like a luxury tour operator)
- `practical_info` (array of exactly 5 associative arrays, each with `icon` [Font Awesome 6 class string], `label` [string], `value` [string]):
  - Valuta: fa-solid fa-coins
  - Lingua: fa-solid fa-language
  - Stagione Migliore: fa-solid fa-sun
  - Fuso Orario: fa-solid fa-clock
  - Visto: fa-solid fa-passport
- `see_also` (array of exactly 4 associative arrays, each with `name` [string], `image` [Unsplash URL], `description` [string — exactly 2 lines of text, rich and evocative]):
  - Use sub-destination suggestions from RESEARCH.md for each continent
- `curiosita` (array of exactly 3 associative arrays, each with `icon` [Font Awesome 6 class], `title` [string], `text` [string — 2-3 sentences of interesting facts]):
  - Choose compelling and surprising facts that enrich traveller knowledge
  - Use varied Font Awesome icons (fa-solid fa-globe, fa-solid fa-mountain, fa-solid fa-star, fa-solid fa-camera, fa-solid fa-leaf, etc.)

Use real-world accurate data for practical_info (from RESEARCH.md reference table). Write Italian copy that reflects the premium, personal, "Lorenzo-is-with-you" brand. Do NOT use placeholder text like "Lorem ipsum" or "Testo qui".

The file must start with `<?php` and the docblock, then assign `$destinations = [...]`. No closing `?>` tag (per PHP best practice).

Use Unsplash source URLs in format: `https://images.unsplash.com/photo-XXXXXXXXXX?w=1600&q=80` — pick actual photo IDs that match the destination (use your knowledge of well-known Unsplash photo IDs for travel destinations).
