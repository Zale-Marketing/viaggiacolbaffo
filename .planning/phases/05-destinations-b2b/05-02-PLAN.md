**File structure (top to bottom):**

```php
<?php
require_once __DIR__ . '/includes/config.php';
require_once ROOT . '/includes/functions.php';
require_once ROOT . '/includes/destinations-data.php';

$slug = $_GET['slug'] ?? '';
$valid_slugs = ['america', 'asia', 'europa', 'africa', 'oceania', 'medio-oriente'];
if (!$slug || !in_array($slug, $valid_slugs)) {
    header('Location: /404');
    exit;
}

$dest = $destinations[$slug];
$page_title = $dest['name'] . ' — Viaggia Col Baffo';
$meta_description = substr($dest['intro_paragraphs'][0], 0, 160);

// Trips for this continent (published only)
$continent_trips = array_values(array_filter(
    get_trips_by_continent($slug),
    fn($t) => $t['published'] === true
));
$has_trips = count($continent_trips) > 0;

$hero_page = true;
require_once ROOT . '/includes/header.php';
?>
```

<task type="auto">
  <name>Task 2: Create api/submit-waitlist.php — cURL webhook endpoint for waitlist form</name>
  <files>api/submit-waitlist.php</files>
  <action>
Create `api/submit-waitlist.php` as a simplified clone of `api/submit-form.php`. Use the skeleton from RESEARCH.md Code Examples (Pattern 4 + cURL Webhook example) exactly.

The file must:
1. Set `Content-Type: application/json; charset=utf-8` header
2. Return 405 on non-POST requests
3. `require_once __DIR__ . '/../includes/config.php'`
4. Accept these POST fields: `nome`, `email`, `telefono`, `destination_slug`, `destination_name` — sanitize each with `htmlspecialchars(strip_tags(trim($_POST[$key])))`
5. Read `WAITLIST_WEBHOOK_URL` using: `$webhook_url = defined('WAITLIST_WEBHOOK_URL') ? WAITLIST_WEBHOOK_URL : '';`
6. If `$webhook_url` is empty: return `{"success":true,"note":"no_webhook"}` and exit (graceful degradation — site works without webhook configured)
7. Build JSON payload with `json_encode($data, JSON_UNESCAPED_UNICODE | JSON_UNESCAPED_SLASHES)`
8. POST via cURL with: CURLOPT_POST, CURLOPT_POSTFIELDS, CURLOPT_RETURNTRANSFER true, CURLOPT_TIMEOUT 10, Content-Type application/json header
9. On cURL error OR non-2xx HTTP: return `{"success":false,"error":"Errore di invio. Riprova o contattaci su WhatsApp."}`
10. On success: return `{"success":true}`

No closing `?>` at end of file.
  </action>
